git-cleanup-branch
==
Small utility to cleanup Git merged branches at both local and remote.

Installation
--
Supported platforms:

- OS X

```sh
git clone https://github.com/ne-sachirou/git-cleanup-branch.git
cd git-cleanup-branch
make
cp git-cleanup-branch SOME_DIR_IN_YOUR_PATH/
```

Usage
--
```sh
cd git_project
git-cleanup-branch
```
